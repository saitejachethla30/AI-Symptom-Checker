# AI Symptom Checker

## 📌 Project Description
This project is an AI-based Symptom Checker that predicts possible diseases based on user input symptoms.

## 🛠 Technologies Used
- Python
- Machine Learning
- Pandas
- Scikit-learn

## 📂 Files
- app.py → Main application file
- Training.csv → Dataset used for training
- requirements.txt → Required libraries

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

## 🎯 Features
- Predicts disease from symptoms
- Simple and easy to use
- Uses machine learning model

## 📊 Dataset
Training dataset contains symptoms and corresponding diseases.
